# Personal-Project-2---Console-RPG

## Description

This program is a simple console rpg made for CMD. 
It is my first gamedev project and serves the purpose of learning the different systems that
one may encounter when developping a game (Inventory system, Combat loop, etc..)

## What does it do?
This console RPG contains a few features:
-Permadeath (Save is deleted upon death)
-Save/Load (Save your progress at any time from the main menu. You will be prompted to load a save file upon launching the program)
-Shop (You can buy items at a shop, which rerolls every now and then)
-Inventory (You can manage your inventory and use consumables directly from it)
-Equipement (You can equip armor, weapons and accessories to help you in your battles)
-Combat (Turn-based combat with many enemies accross different maps)

## How to use?
1) Clone the repository
2) Open the project in your IDE of choice (Visual Studio recommended)
3) Compile and run the program
4) Choose a slot to create a new save
5) Have fun!
